What is the FNFTA?

The First Nations Financial Transparency Act requires First Nations to publish audited financial statements and Chief & Council remuneration schedules on ISC's website.

⌕

What does OpenBand show?

OpenBand displays parsed remuneration rows when available, keeps pending filings visible, and links back to the original ISC PDF for every filing.

!

Data availability

Some PDFs are posted but not parsed yet. When a newer year is empty, OpenBand opens the newest parsed year by default and labels the pending year.

Source First

Every result starts from an Indigenous Services Canada FNFTA filing page or PDF. The source link stays visible so readers can verify the row-level data.

Machine Parsed

Rows are extracted from PDF tables, PDF text, or an AI fallback for harder layouts. The extraction method is shown in the source note.

Audit Trail

The public audit report tracks parsed filings, pending posted filings, missing Saskatchewan names, and parser statuses after each run.

Use Carefully

OpenBand is designed for leads, comparison, and public accountability. Confirm exact figures against the original filing before quoting them.
