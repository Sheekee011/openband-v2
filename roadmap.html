OpenBand Roadmap

Trust first. Analysis next.

OpenBand is being built as public records infrastructure: searchable FNFTA filings today, deeper comparison and research tools once the data is strong enough.

What This Means

The first milestone is becoming the cleanest Saskatchewan FNFTA remuneration dataset online. After that, OpenBand can grow into an analysis layer for population comparisons, trends, summaries, exports, heatmaps, and alerts.

Phase 1

Build Trust

Make the public records easy to find, compare, and verify.

  • Parse all Saskatchewan bands with posted remuneration filings.
  • Reduce pending filings without sacrificing accuracy.
  • Keep every row linked to the source ISC PDF.
  • Show audit status publicly.
  • Label parsed, pending, not posted, and review-needed filings clearly.
Phase 2

Build Analysis

Turn reliable records into tools for researchers, journalists, policy analysts, and communities.

  • Compare bands by population and fiscal year.
  • Show compensation trends and historical changes.
  • Create neutral automated summaries.
  • Add searchable categories and stronger exports.
  • Build alerts for new filings and major changes.

Phase 2 Feature Set

These tools should only launch once the dataset is reliable enough that the analysis does not mislead people.

Population Comparison

Compare remuneration against sourced population data, with clear labels for population date and source.

Trends

Show year-by-year changes for total paid, remuneration, expenses, travel, and council size.

Summaries

Generate plain-language filing summaries that stay neutral and link back to source records.

Search Categories

Filter by fiscal year, treaty area, population range, payment range, status, and extraction method.

Heatmaps

Use visual maps and grids to find patterns in compensation, pending filings, and future spending categories.

Exports

Provide full CSV and JSON exports with source URLs, parse status, extraction method, and generated date.

Alerts

Notify users when filings are posted, pending data is parsed, or major year-over-year changes appear.

Historical Profiles

Build a timeline for each First Nation showing parsed years, pending years, and changes over time.

Trust rule: OpenBand should not imply wrongdoing from high numbers alone. The goal is to make public records easier to access, verify, and understand.
